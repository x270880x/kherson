# Кадастровая карта — Херсонская область

Журнал работ по проекту. Дата старта: **17.06.2026**.

## Что это за проект

Интерактивная кадастровая карта Херсонской области на статическом сайте (GitHub Pages).

- **Репозиторий:** https://github.com/x270880x/kherson
- **Живой сайт:** https://x270880x.github.io/kherson/
- **Источник данных:** клон публичного `traidoptmarketnew/kadastr-nyzhni-sirohozy`, отфильтрован до Херсонской области.
- **Стек:** статический `index.html` + Leaflet 1.9.4 + Leaflet.markercluster + SheetJS (xlsx, экспорт).
- **Данные:** встроенный JSON (сводки 337 сельсоветов) + ленивая подгрузка `parcels/*.geojson` и `councils/*.json` (675+675 файлов).

## Основные сделанные блоки

### Имена по Яндексу + выравнивание подписей на центр села + навигация беты (2026-06-21)
- **Эталон имён НП — Яндекс.Карты** (РФ-наименования «на сейчас»). vkursi имён не отдаёт (поля `np`/`ra` пустые — проверено на проксе), OSM/КОАТУУ дают укр./декоммунизированные имена. Ключ Геокодера — `~/.yandex_geocoder_token` (600); тулинг `.wip/refetch/yandex_audit.py`, `pos_align.py`, `geo_audit.py`.
- **«Двойник Ивановки» раскрыт:** совет `6522985900` — это **Трохимівка** (Трофимовка), ошибочно назван по громаде. Аналогично — ещё ряд советов, названных по чужому/соседнему селу.
- **93 совета переименованы/выровнены:** 10 (web+КОАТУУ) + 65 (Яндекс, объект <2 км от жилого кластера) + 12 (доппроверка обоих языков) + 6 (многоточечная сверка ядра 02.01: `Зелене→смт Верхній Рогачик`, `Старолук'янівка→Червона Поляна`, `Остапенка→Волинське`, `Зелений Гай→Українське`, `Дружелюбівка→Василівка`, `Добросілля→Великі Копані`). Метод: reverse/forward-геокод жилого кластера + КОАТУУ-код рады (= её центр) + web (decentralization/Вікіпедія). 276 имён совпали с Яндексом сразу.
- **Все 372 подписи выровнены на центр села** (`pos_align.py`): reverse-геокод кластера → точка села Яндекса (350) либо жилой кластер. **142 подписи стояли >1 км в полях** — теперь по центру. Все точки в пределах области (0 выбросов).
- **Верхній Рогачик** переехал из ручной точки в данные (совет 6521555100, был «Зелене»): в map-beta убрана ручная `feats.push` + район 215 из skip авто-центра (теперь центр выбирается из данных как смт).
- **Навигация беты (по фидбеку):** старт = вся область (`START_ZOOM = fz+1`); **зум-аут максимум 2 нажатия минуса** (`minZoom = START_ZOOM − 0.70`, шаг 0.35); пан **заблокирован на старте и 2 отдалениях**; при зуме-ин пан **только в пределах области** (`maxBounds = PAN_BOUNDS [[31.48,45.72],[35.15,47.65]]` = края парцел +0.05° = «до краёв +1 см, не дальше»).

- **Сверка с источником** (выборка 1800): геометрия (IoU 0.996), кадастры, площадь, форма собственности, назначение — совпадают ~100%.

- **Добор пропущенных:** инверсная сверка (172k тайлов кэша vs база) нашла 9 463 участка в источнике, которых не было. Из них 2 877 дописаны в существующие советы; **7 сёл, отсутствовавших целиком**, досканированы flood-fill'ом и созданы заново — **Старосілля, Новопавлівка, Нова Калуга, Біла Криниця, Білоусове, Киселівка, Посад-Покровське** (~10 055 уч.; в кэше было лишь ~5.5к — flood-fill добрал полностью). 40 мелких суб-зон влиты в родительские советы.

### Перекачка/починка геометрии из снапшота (2026-06-20)
- Полная сверка геометрии участков с тайловым снапшотом `kadastrova-karta.com.ua` (supabase `tiles-proxy`, MVT z16). Атрибутный API `back.geoserver.click/api/landuse` по оккупированному Херсону отдаёт 404 (данные изъяты из офиц. ДЗК) — атрибуты оставлены из нашей `councils/*`, обновлена только геометрия.
- **Починено 9 501** обрезанных/битых участков: реальный контур собран склейкой кусков из соседних тайлов (тайлы режут по своим границам). Проверка по площади `a` — медиана ошибки 0.0%, p90 ~0.2%. Пример: участок был 0.39 га (обрубок) → стал 10.2 га (полный).
- **Добавлено 25 756** участков, которых не было в базе (найдены flood-fill'ом по префиксу кадастра; целые советы были почти пустыми, напр. многие с 1 участком). Геометрия-only, форма собственности «не определено» (`ot=""`), пометка `src:"snapshot2026"`. Наложения на существующие 0%.
- Геометрия снапшота смещена ровно на +5.00 м по широте — выровнено к нашему фрейму (после поправки IoU с существующими хорошими участками 0.997).
- Хорошие 521 088 участков не тронуты. Кадастры с битым байтом (`\x16`) нормализованы; стрэй-файл-дубликат схлопнут. Итого ~556 355 участков (было 530 600).

### Данные и регион
- Полностью импортирован и развёрнут проект (app + 675 файлов участков/советов), включён GitHub Pages.
- Карта ограничена **только Херсонской областью** (337 советов), заголовочная статистика пересчитана.
- Район РФ для каждого совета **выведен из кадастрового кода** (геокодирование координат → 5 районов: Каховский, Генический, Скадовский, Херсонский, Бериславский). Таблица «префикс кода → район» в коде.

### Производительность
- Canvas-рендерер + `smoothFactor`, выгрузка полигонов прошлого села, делегирование событий и ленивые попапы (убран O(n²) при открытии села).
- `defer` для тяжёлого SheetJS (xlsx) — не блокирует загрузку.
- Минификация герба.

### Карта и маркеры
- Полупрозрачные маркеры + кластеризация по зуму (markercluster); в кластере — сумма участков.
- Старт карты на ~10% ближе (`zoomSnap: 0`).
- Тултип: русское название села крупно + район мелким; чёрный жирный текст на белом.
- Клик по селу → приближение к его участкам ×2; зум без анимации (тёмная схема не мерцает).
- При фильтре на карте остаются только подходящие участки, остальные полигоны скрываются.
- Цвета форм собственности едины везде (карта, фильтр, деталь, попап).
- Переключатель слоёв «Схема тёмная/светлая/Спутник» свёрнут в иконку-шестерёнку (открытие по клику, закрытие по клику вне).

### Язык
- Интерфейс переведён на русский (заголовок, формы собственности).
- На карте приоритет русскому: названия сёл, формы собственности; украинский скрыт (категории/имена арендаторов оставлены как юр. данные).
- Префиксы типов НП переведены: `смт → пгт`, `м. → г.`
- Метки: «Размежевание (РФ кадастр)», «Категория земель (УКР)», «Арендатор (УКР)».

### Шапка
- Добавлен герб Херсонской области (РФ, проект 2022, public domain с Wikimedia).
- Заголовок в одну строку, придвинут к гербу; блок статистики — симметричные равные колонки с разделителями.

### Подборка участков (бывш. «Корзина»)
- Переименована в «Подборка» (иконка 📌).
- Рамочное выделение: Shift+перетаскивание мышкой добавляет все участки в области.
- Правая панель показывает список выбранных (с площадью каждого, итоговым числом и суммарной площадью).
- Сброс всей подборки: ✕ в заголовке слева и кнопка «✕ Сброс» в правой панели.

### Правая панель (деталь села)
- Выдвижная сворачиваемая панель (кнопка «›» сворачивает, язычок «‹» разворачивает).
- Название: русское крупно + «Район РФ (укр. название)» мелким снизу.
- Блок «Форма собственности» — горизонтальная полоса в цветах.
- На мобиле не открывается (карта на весь экран).

### Мобильная версия (≤768px)
- Компактная шапка с переносом, карта на весь экран.
- Фильтры — выезжающая слева шторка по кнопке-бургеру ☰ (на месте зум-кнопок, слева вверху карты).
- Зум +/− скрыт (зум пальцами).

### Приватность / индексация
- `noindex` мета + `robots.txt` (Disallow: /) — закрыто от Google/Яндекса (только для kherson и buhta; SplitCam-сайты не трогали из-за SEO).
