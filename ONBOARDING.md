## Как запустить (preview)
Сервер настроен в `~/.claude/launch.json` под именем **`kherson`** (python `serve.py`, порт **8773**, корень `~/Documents/kherson`).
1. `preview_start` с `name: "kherson"` (НЕ через Bash).
2. Проверять: `preview_eval` (состояние), `preview_screenshot` (вид), `preview_console_logs` (ошибки).
3. **Важно:** preview-окно бывает узким (мобильная ширина). Для десктоп-проверки `preview_resize` на 1440×900; для мобильной — preset `mobile` (375).
4. `preview_click` ненадёжен для Leaflet-контролов (шестерёнка) — поведение проверять через эмуляцию событий (`dispatchEvent`/`map.fire`), это эквивалент реального клика.
5. **Бета** открывается по `http://localhost:8773/map-beta.html`. Гоча: иногда свежий preview-инстанс **не рисует MapLibre** (пустой экран, `map.getStyle()` пуст) — это баг инстанса, не кода (заведомо рабочая версия тоже пустая). Лечится `preview_stop` + `preview_start` (новый сервер). Однопоточный `serve.py` медленно отдаёт тяжёлые тайлы беты — для оценки скорости смотреть живой сайт (CDN), не локалку.
6. ⚠️ **`serve.py` НЕ в репозитории** (как и `.wip/`). При свежем клоне preview не поднимется, пока не положишь `serve.py` рядом.

## Структура
- **`index.html`** (~506 КБ, 518 486 байт) — основная карта (Leaflet) целиком в одном файле:
  - `<style>` в `<head>` (включая `@media (max-width: 768px)` — мобильная версия);
  - `<script id="data">` (строка ~381, минифицированный JSON `separators=(',',':')`) — сводки советов (code, name, lat, lon, region, total, area, by_ownership, top_owners/renters/categories, lease_years) + `stats`, `filters`, `renter_councils`. `STATS = DATA.stats`. ✅ **Пересобран 2026-06-21** под 372 совета (Херсон-only): `stats` total_parcels 613471 / councils 372 / total_area 2162361.0 / `regions` только `{Херсон}`. Регенерация — агрегация `councils/*.json`: `total`=len, `area`=Σ`a`, `by_ownership`=Counter `ot`, `top_categories`=top-5 `ct`, `top_renters`=Counter `r` (desc), `lease_years`=Counter `ey`; имена из `councils-index.json` `n` (там 22 ручных переименования, в файлах советов — «сырые»); глобальные `stats`/`filters` (categories=top-20, renters=все)/`renter_councils`. Генератор был разовый (не в репо) — при изменении данных пересобрать тем же способом.
  - основной JS в конце `<body>`.
- **`parcels/*.geojson`** (**372 файла, 613 471 участок**) — геометрия участков (в properties только `cadnum`), ленивая подгрузка по коду совета. Не-херсонские удалены. **2026-06-20/21:** геометрия сверена со снапшотом kadastrova-karta.com.ua (починены обрезанные, добавлены пропущенные, `src:"snapshot2026"`); добавлены целиком **Нововоронцовский** (КОАТУУ 65241) и **Высокопольский** (65218) районы с атрибутами из vkursi (`src:"vkursi2026"`). См. CHANGELOG и [[project_kherson_refetch]]. Тулинг — `.wip/refetch/` (gitignored).
- **`councils/*.json`** (**372 файла**) — атрибуты участков совета (`c`,`a`,`o`,`r`,`ot`,`ct`,`mz`…), ленивая подгрузка. Не-херсонские удалены. Новые участки — стабы либо обогащены vkursi.
- **`councils-index.json`** (**372 записи**) — на запись: `c` (код КОАТУУ), `n` (имя «укр (рус)»), `lat`, `lon`, `o:{p,k,g,n}` (счётчики участков по форме собственности), `total`, `ha`. Источник для беты: клик/поиск/подписи сёл/таблички наведения.

- **`tiles/{z}/{x}/{y}.pbf`** (**~149 МБ, 13 076 файлов, zoom 6–14**) — векторные тайлы всех 613 471 участка. Единственное свойство — `o` (форма собств. p/k/g/n) для раскраски; кадастр/площадь/данные участка берутся при клике из оригинальных `parcels/`+`councils/` (point-in-polygon), не из тайлов. Лимит тайла 2 МБ + только `o` → прореживание идёт только на нижних зумах (z6–8, где область умещается в пару тайлов); с z9 участки не прореживаются. Флаги: `-Z6 -z14 -y o --drop-densest-as-needed --maximum-tile-bytes 2000000 --no-tile-compression --force` (слой `parcels`). Запускать при изменении данных участков/советов; выше z14 — overzoom на клиенте.

## `map-beta.html` — бета-карта (MapLibre GL JS 4.7.1)
GPU-рендер всех участков из `tiles/` (цвет по `o`; `fill-antialias:false`, `fadeDuration:0`, `fill-opacity:0.45`). Свои кнопки +/− (`#zoomctl`, `easeTo ±0.35`, 320мс ease-in-out). Последний коммит беты — **bd3fe9b9**.

### Стартовый вид
- `KHERSON_BOUNDS = [[31.80,45.90],[35.10,47.45]]` (запас под края уже в bounds).
- Конструктор: `bounds: KHERSON_BOUNDS`, `fitBoundsOptions:{padding:0}` (область до краёв кадра), `minZoom:6`, `maxZoom:19`, `fadeDuration:0`.
- `setupStart()` (по `idle` + таймеры 300/800мс, идемпотентно): `fz = cameraForBounds(KHERSON_BOUNDS,{padding:0}).zoom`; **`START_ZOOM = fz+1`** (вся область в кадре); `jumpTo({zoom: START_ZOOM})`; **`setMinZoom(START_ZOOM − 0.70)`** — зум-аут **максимум 2 нажатия минуса** (шаг 0.35), дальше блок.
- `updateZoutBtn()` гасит кнопку «−» (opacity .3, pointer-events:none) при `zoom ≤ minZoom+0.05`.
- `applyPanLock()` (по `zoomend`): **пан заблокирован на старте и 2 отдалениях** (`zoom ≤ START_ZOOM+0.05` → `dragPan.disable()` + `setMaxBounds(null)`, чтобы зум-аут до minZoom не блокировался); при **зуме-ин** → `dragPan.enable()` + **`setMaxBounds(PAN_BOUNDS)`** = пан только в пределах области. Клик/поиск приближают программно — от пан-лока не зависят.
- `PAN_BOUNDS = [[31.48,45.72],[35.15,47.65]]` — реальные края парцел Херсонщины +~0.05° («до краёв области +1 см, не дальше»). `maxBounds` не трогает зум (его floor ниже minZoom).
- Source `parcels` `minzoom:6, maxzoom:14`; map `minZoom:6` → **вся область рендерится участками с первого кадра на любых экранах** (костыль `minZoom:8`, который обрезал область на узких, снят — тайлсет пересобран `-Z6`).

### Слои подписей (источник `omt` = OpenFreeMap vector + наш `councils` geojson; глифы self-hosted)
- **`lbl-water`** — гидронимы, `name:ru` coalesce, синий.
- **`lbl-place`** — OMT-НП, фильтр `class in [city,town]` (минус наши имена `ourNames`). Приглушение чужих областей: `isOut = name ∈ OUT_OF_REGION=[Армянск/Армянськ/Красноперекопск/Красноперекопськ] OR !within(hull)`, где `hull` = выпуклая оболочка (convexHull, алг. Эндрю) точек советов. Чужие НП: `text-opacity 0.25`; `text-size = interpolate(zoom, 6→case(isOut,8.8,13.2), 11→case(isOut,12,18))` — чужие в **~1.5× мельче**. `within(hull)` работает в layout `text-size` в MapLibre 4.7.1.
- **`lbl-state`** — соседние области (`class=='state'`, минус «Херсонская область»), вдвое мельче, `text-opacity 0.5`.
- **`lbl-councils`** — наши сельсоветы (пгт/сёла, не центры), фильтр `!ctr`. Конструируется на `minzoom:9`, но в `setupStart()` переставляется на **`fz+1.6`** → **на обзоре сёл нет, появляются при приближении**. `symbol-sort-key` по `-total`, collision.
- **`lbl-centers`** — райцентры, `minzoom:7`, БОЛД ×1.5, `text-allow-overlap:true`, перед `lbl-place`. Центр района = главный (макс. участков) урбан-НП. **Из авто-выбора центра исключены 3 района** — `'город Херсон'`, `'Верхнерогачикский район'`, `'Великоалександровский район'` (префикс 209): их центры добавлены вручную (★Херсон 32.6169/46.6354; Верхний Рогачик 34.3611/47.2647; Великая Александровка 33.2786/47.2536). После исключения 209 село **Борозенське (6520981000) больше НЕ промоутится в райцентр** — стало обычным советом (`lbl-councils`, появляется при зуме). `BIG_TOWNS={6510700000,6510400000}` (Нова Каховка, Каховка) видны на обзоре как `big`.
- **`centers-hit`** — прозрачная круг-хитзона вокруг райцентров (`minzoom:7`, `circle-opacity:0`, радиус 22→11 по зуму) — крупнее подписи, легче попадать мышкой.

### Слой межевания `razmezh` (`razmezh.geojson`) — `setRazmezh(mz)`, фиолетовый `#c050ff`
- **`mz='yes'` (только размежёванные):** ТОЛЬКО розовый ободок — `razmezh-fill` скрыт, `razmezh-line` виден (`line-width` 0.7→2.4); основной слой `parcels-fill`/`parcels-line` тоже скрыт. **Как в оригинале.**
- **`mz='no'` (только без межевания):** слой межевания полностью скрыт.
- **`mz=''` (по умолчанию):** лёгкая заливка `razmezh-fill` (`fill-opacity` z8:0.35→z13:0, гаснет к z13) + ободок `razmezh-line`.
- **Фильтр межевания работает на ЛЮБОМ зуме** через глобальный слой `razmezh` — без порога z13 и подсказки «Приблизьте».

### Клик-зум
- **Клик по району** (хитзона `centers-hit` ИЛИ `lbl-place` `class∈[county,district]`): `fitBounds` по **ФАКТИЧЕСКОМУ bbox всех советов района** (ближайший совет → префикс `code[2:5]` → bbox; `±0.05°` запас, `padding:22`, 1600мс) → весь район в кадре по максимуму. Фолбэк `flyTo zoom:9`.
- **Клик по селу** (`lbl-place` village/hamlet, `lbl-councils`): `drillVillage()` → `DRILL_SEQ=[4,2,1]` (`fitNearestN`): 1-й клик кадрирует **~4 села**, повторные клики ближе (2→1). Клик по другому селу/району сбрасывает прогрессию.
- Guard в `click parcels-fill`: клики по любой подписи НП → зум, а не плашка участка; при >10 видимых НП участки не кликабельны.

### Наведение и попап
- **Табличка наведения** (`hoverPop`, `pointer-events:none`): hover по `lbl-place`/`lbl-councils` → `ownTableHTML(c)` ближайшего совета (`nearestCouncil`, ~22км) — счётчики форм + всего/га. Hover по `centers-hit` → `raionTableHTML(rname)` — агрегат по всему району.
- **Попап участка** (`popupHTML`): УКР-кадастр с копированием; совет + площадь; **район РФ** (`raionOf`/`RAION_BY_PREFIX`); собственник `o`; форма собственности полным текстом (`OTC`); **«Назначение»** (`ct`) и **«Использование»** (`use`) отдельными строками; блок «🇷🇺 Межевание сделано» с РФ-номером (`mz`) + копирование. Точный участок ищется point-in-polygon по `parcels/<code>.geojson` + ленивая `councils/<code>.json`. `prefetchNear()` прогревает кэш на hover.

### Тонкие фильтры (`fine` = {mz,ct,ly,r})
- Категория/год аренды/арендатор требуют поучастковых данных → порог **`FINE_MINZOOM=13`**: при z≥13 `parcels-fill/line` скрываются, оверлей `fine` показывает подходящих из подгруженных советов (до 14); при z<13 — подсказка «Приблизьте, чтобы применить фильтр».
- **Межевание (`f-mz`) работает на ЛЮБОМ зуме** через `setRazmezh` — БЕЗ порога/подсказки.
- `fineUpdate()` сериализован, перезапуск на `moveend`. Точное имя арендатора (`RENTC[fine.r]`) → перелёт `fitBounds` к его советам.

### Прочее
- **Поиск** (`#q`): село (рус/укр, `ruName`) / кадастр (`CAD_RE /\d{10}:\d{2}:\d{3}:\d{4}/`) / арендатор; Enter по кадастру → `flyCad` (зум 17 + попап).
- **Подборка** (`basket` Map): Shift+клик / Shift+рамка → подсветка жёлтым, счётчик, **CSV-экспорт** (`podborka.csv`, `;`, BOM).
- **Титул региона** (`#region-title`/`.rt-inner`, CSS-оверлей): герб + «Херсонская область» по центру; при `zoom > TITLE_AWAY_ZOOM=8.6` уезжает в правый верхний угол (`translate(42vw,-44vh) scale(.62)`, `transition .8s`) и остаётся; цвет под подложку.

### Источники данных
- **Геометрия:** тайловый снапшот `kadastrova-karta.com.ua` → supabase `tiles-proxy` (MVT, z16, слой `tile_land_polygons`, свойство только `cadnum`). Полный контур участка = склейка кусков из соседних тайлов (тайлы режут полигон по своим границам, flood-fill-объединение). **Поправка +5.00 м по широте** выравнивает к нашему фрейму (IoU с хорошими участками ~0.997). Ядро — `kref.py` (содержит supabase anon-ключ, exp 2078 — сторонний ключ снапшота).
- **Атрибуты:** `vkursi-proxy` (агрегатор vkursi.com) — форма собственности / назначение / использование / площадь / село / район; батч ≤10 кадастров.

- ⚠️ **Имена собственников-физлиц НЕ скрапим** — доступны только через платный поучастковый витяг ДРРП + это персональные данные. Поле собственника в попапе остаётся юр-данными/арендаторами, НЕ ФИО владельцев. **Это сознательное правило, не TODO.**
