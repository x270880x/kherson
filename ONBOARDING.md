## Как запустить (preview)
Сервер настроен в `~/.claude/launch.json` под именем **`kherson`** (python `serve.py`, порт **8773**, корень `~/Documents/kherson`).
1. `preview_start` с `name: "kherson"` (НЕ через Bash).
2. Проверять: `preview_eval` (состояние), `preview_screenshot` (вид), `preview_console_logs` (ошибки).
3. **Важно:** preview-окно бывает узким (мобильная ширина). Для десктоп-проверки `preview_resize` на 1440×900; для мобильной — preset `mobile` (375).
4. `preview_click` ненадёжен для Leaflet-контролов (шестерёнка) — поведение проверять через эмуляцию событий (`dispatchEvent`/`map.fire`), это эквивалент реального клика.

## Структура
- **`index.html`** (~788 КБ) — весь сайт в одном файле:
  - `<style>` в `<head>` (включая `@media (max-width: 768px)` — мобильная версия);
  - `<script id="data">` — встроенный JSON: сводки 337 советов (code, name, lat, lon, region, total, area, by_ownership, top_renters/owners/categories, stats, filters);
  - основной JS в конце `<body>`.
- **`parcels/*.geojson`** (675 файлов, ~282 МБ) — геометрия участков (в properties только `cadnum`), ленивая подгрузка по коду совета.
- **`councils/*.json`** (675 файлов, ~220 МБ) — атрибуты участков совета (`c`,`a`,`o`,`r`,`ot`,`ct`,`mz`…), ленивая подгрузка.
- **`gerb.svg`** — герб Херсонской области (РФ, проект 2022).
- **`robots.txt`** — `Disallow: /` (+ `noindex` мета в `index.html`).
- **`CHANGELOG.md`**, **`README.md`**.

## Ключевые места в коде (index.html)
- **Карта:** Leaflet 1.9.4 + markercluster, `preferCanvas: true`, `zoomSnap: 0`, `maxZoom: 19`. Базовые слои: `baseDark` (OSM + CSS-инверсия `.map-tiles-dark`), `baseLight`, `baseSat` (Esri). По умолчанию — тёмный.
- **Слои:** `layersCtrl` свёрнут в иконку-шестерёнку (SVG); открытие по клику, закрытие по mousedown вне (hover отключён).
- **Маркеры/кластеры:** `buildMarkers()` (вызывается при каждой смене фильтра) → `clusterGroup`. В кластере — сумма участков (`m.pc`).
- **Полигоны:** `loadCouncilPolygons(code)` (lazy, кэш `polygonLayers`, показывает только текущее село — `showOnlyPolygons`). `parcelStyle(f, council)` — **прячет участки, не подходящие под фильтр** (`hasAnyFilter()` + `parcelMatchesFilters`). При смене фильтра — `restyleFilteredPolygons()`.
- **Фокус на село:** `focusCouncil(code, closer)` — `showDetail` + загрузка контуров + **плавный `flyTo`** (duration 0.6с) на целевой зум (вписать участки с учётом отступов + кратность `closer = log2`, центр смещён под правую панель). Клик по маркеру = `focusCouncil(code, Math.log2(1.5))` (×1.5). Поиск/список — `focusCouncil(code)` (без доп. приближения). Анимация в preview не проигрывается (rAF на паузе в неактивной вкладке) — проверять мгновенным `setView`.
- **Район РФ:** `RAION_BY_PREFIX` (ключ = `code[2:5]`) → `raionOf(code)`. Получен геокодированием координат (OSM, ru). 5 районов.
- **Названия:** `nameParts(name)` → `{ru, ua}` (рус из скобок + перевод префикса смт→пгт, м.→г.); `ruName(name)` (только рус); `detailTitle(c)` (рус крупно + «район (укр)» мелким).
- **Фильтры и подсчёт:** `getFilteredStats(council)` — точный подсчёт, если `council.parcels` загружены (`parcelMatchesFilters`), иначе приближённый по агрегатам сводки. `buildMarkers()` прячет кружок, если под фильтром `count` не `> 0` (ловит 0 и NaN). Межевание (`filterMz`, РФ кадастр) нет в агрегатах сводки — отдельная карта **`MZ_COUNT`** `{код: число_размежёванных}` (105 советов, рядом с `getFilteredStats`); пересчитывается из `councils/*.json` (участки с непустым `mz`).
- **Формы собственности:** `OWN_COLOR`/`ownColor` (цвета: Частная #4caf7d, Коммунальная #e8c547, Государственная #4a9eff, Не определено #e05252); `OWNERSHIP_RU`/`ruOwn` (укр→рус только для отображения, данные не меняются); `OWN_SHORT`/`ruOwnShort`.
- **Подборка (basket):** `basket` (Set кадастров), `basketUpdate()`, `renderBasketPanel()` (правая панель со списком, площадями, итогами). Рамочное выделение: Shift+drag (`setupBoxSelect`). Сброс: кнопка «Очистить подборку» (`btn-basket-clear`, левая панель) и «✕ Сброс» в правой панели.
- **Правая панель** `aside.right` — выдвижная (классы `.open`), кнопки «›» свернуть / «‹» развернуть. На мобиле при выборе села НЕ открывается.
- **Мобайл (≤768px):** карта на весь экран; фильтры — выезжающая слева шторка `aside.left.open` по кнопке `#mobile-filters-btn` (☰, слева вверху карты); зум +/− скрыт.

## Соглашения (как тут принято работать)
- **После каждой осмысленной правки — `git commit` + `git push`** (origin main), коммитить как `x270880x <x270880x@gmail.com>`. Сайт на Pages обновится сам.
- **Перед push — проверять в preview** (eval/скриншот, console на ошибки).
- **Язык:** на карте приоритет русскому (укр. скрыт). НО категории земель и имена арендаторов/собственников оставлены на украинском — это юр. данные/имена (помечены «(УКР)»). Перевод 67 категорий — отдельная задача (не сделана).
- **Точность данных:** правки UI/отображения делаются в `index.html`, 675 файлов данных не трогаем без явной задачи.
- **НЕ ставить noindex на SplitCam-репозитории** — там активный SEO.

## Открытые вопросы / не сделано
- **Поиск по РФ-кадастру** — формат распознаётся, но РФ-номеров в данных нет (искать нечего). Нужен источник (файл / НСПД).
- **Оптимизация данных** — можно срезать точность координат в `parcels/*.geojson` 6→5 знаков (~15% размера); не делалось (огрубляет границы, правка 675 файлов).
- **Подписи на тайлах** — OSM на украинском; русские требуют другого поставщика (Яндекс/2ГИС с ключом) или векторной карты.
- **PIN/вход на сайт** — обсуждали; реально закрыть можно только через Cloudflare Access (статический GitHub Pages серверной авторизации не умеет).
- **Район** выведен из кода геокодированием — возможны единичные неточности на границах (правятся вручную в `RAION_BY_PREFIX`).
